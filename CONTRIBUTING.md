# Contributing

Thank you for improving Online Harassment, Blackmail & Impersonation Response.

## Good contributions

- Correct outdated official links
- Clarify safety steps without promising outcomes
- Improve accessibility and plain-language explanations
- Add translations that preserve the original meaning
- Cite primary sources such as regulators, law-enforcement agencies, standards bodies, and official platform help centers

## Pull request checklist

- [ ] The change contains no real victim or case data
- [ ] Claims are supported by authoritative, current references
- [ ] Instructions are lawful, defensive, and safety-first
- [ ] The change does not advertise paid recovery, hacking, tracing, or retaliation
- [ ] Links use HTTPS and point directly to the cited source

By contributing, you agree that maintainers may edit submissions for safety, clarity, and privacy.
