# Security and privacy policy

## Do not submit case evidence

This repository publishes general guidance only. Do not place personal data, credentials, financial records, private media, victim information, or active-case evidence in issues, discussions, commits, or pull requests.

## Reporting a problem with this repository

For a broken official link, unsafe instruction, privacy concern, or security issue in the documentation, open a minimal issue that contains no sensitive data. Describe the affected file and section without including real case material.

## Active incidents

Use the official reporting and recovery routes listed in [README.md](README.md). Repository maintainers cannot investigate crimes, recover accounts, trace people, or receive evidence on behalf of an authority.
